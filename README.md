# Frontend Job Portal UI Module

React frontend module for the Online Recruitment and Job Application Management System.

This folder is ready to open directly in VS Code and submit as the `frontend-job-portal` module for the group project.

## Features

- Job listing page with responsive job cards
- Job details page with job description, skills, salary, location, and company information
- Search jobs by title, company, keyword, category, skill, or location
- Filter jobs by location, category, employment type, and salary range
- Pagination, loading, and empty-results states
- UI-only Apply Now action on the job details page

## Scope

This is a frontend-only module. It does not include authentication, backend APIs, database logic, admin dashboard screens, application forms, or resume upload features.

## Tech Stack

- React
- React Router
- Bootstrap 5
- Vite

## Run Locally

```bash
cd frontend-job-portal
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Build

```bash
npm run build
```
